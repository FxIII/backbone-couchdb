backbone-couchdb by janmonschke

A CouchDB connector for Backbone.js with support for real time changes. Check out the annotated source and the example couchapp.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/janmonschke/backbone-couchdb

get the source code on GitHub : janmonschke/backbone-couchdb
